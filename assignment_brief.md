# 미니프로젝트: KOBIS 박스오피스로 보는 한국 영화 시장 (2020-2026)

## 프로젝트 개요

영화진흥위원회(KOBIS) 박스오피스 데이터 7년치(2020~2026)를 분석해서, 한국 영화 시장에서 일어난 변화와 흥행 패턴을 직접 찾아내는 개인 프로젝트입니다.


## 학습 목표

이 프로젝트를 끝내고 나면 여러분은 다음을 할 수 있어야 합니다.

1. 실제 데이터에서 결측치·이상치를 진단하고 처리
2. `groupby`, `pivot_table`로 의미 있는 집계 만들기
3. 시계열 데이터에서 월·연도 정보 추출하기
4. Matplotlib·Seaborn으로 메시지가 있는 그래프 그리기
5. 분석 결과를 비전공자도 알아들을 수 있는 한 줄로 정리하기


## 데이터 설명

배포 파일: `kobis_boxoffice_2020_2026.csv` (약 3MB, 29,724행)

| 컬럼 | 의미 |
|---|---|
| `조회연도` | 박스오피스 집계 연도 (2020~2026) |
| `순위` | 해당 연도 박스오피스 순위 |
| `영화명` | 영화 제목 |
| `개봉일` | 영화 최초 개봉일 (결측 다수) |
| `매출액` | 해당 연도 매출액 (원) |
| `매출액 점유율` | 0~1 사이 비율 |
| `관객수` | 해당 연도 관객수 (명) |
| `스크린수` | 상영 스크린 수 |
| `상영횟수` | 상영 횟수 |
| `대표국적` | 대표 제작 국가 |
| `국적` | 제작 국가 (다국적이면 여러 개) |
| `배급사` | 배급사 |

**알아둘 점 두 가지**
1. **2026년은 5월 24일까지의 데이터만** 들어 있습니다 (반쪽짜리 한 해). 연도별 비교에 포함할지 직접 정하세요.
2. **같은 영화가 여러 연도에 등장할 수 있습니다**. 예: 2024년 12월 개봉작은 2024년·2025년 박스오피스에 따로 기록됩니다. 분석 목적에 따라 다르게 다뤄야 합니다.

## 데이터 정제 가이드

분석 시작 전에 다음을 확인하세요. 어떻게 처리할지는 본인이 정하되, **마크다운 셀로 그 이유를 반드시 설명**해야 합니다.

- 관객수가 `0`인 행이 꽤 있습니다. 분석에 포함할까요, 뺄까요?
- `개봉일`이 결측인 영화가 약 1만 1천 개. 무엇이 빠져 있을까요? (힌트: 단관 상영, 예술관, 시네마테크 재상영작)
- 어떤 임계치(예: 관객수 ≥ 1만)로 필터링하는 게 합리적일까 직접 정해 보세요.

좋은 정제의 기준: **분석 결과가 임의로 흔들리지 않아야** 합니다.

## 분석 질문

각 질문에 대해 **(1) 분석 코드 → (2) 그래프 → (3) 한 줄 해석** 순서로 답합니다.

### 질문 1. 한국 영화 시장은 코로나 이후 어떻게 변했나요?
- 조회연도별 총 관객수 변화를 선 그래프로
- 2026년 포함 여부를 직접 판단

### 질문 2. 어느 달에 개봉한 영화가 가장 흥행하나요?
- 개봉월별 평균 관객수 비교
- `pd.to_datetime` 이미 적용된 컬럼에서 `.dt.month` 추출

### 질문 3. 한국 영화와 외국 영화의 점유율은 연도별로 어떻게 변했나요?
- 한국 / 미국 / 일본 / 기타 정도로 그룹핑
- `pivot_table` 활용, stacked bar 또는 line으로 시각화

### 질문 4. 스크린 효율성 — 적은 스크린으로 흥행한 영화는?
- `관객수 ÷ 스크린수`로 효율 계산
- 산점도로 전체 분포 + Top 10 표

### 질문 5. 박스오피스의 불평등 — 상위 10편이 전체 시장의 몇 %를 차지하나요?
- 연도별 상위 10편의 관객수 합계 ÷ 그 해 전체 관객수
- 보너스: 상위 1편만 따로 보면 어떻게 다른가?

### 질문 6 (자유). 본인이 궁금한 것
앞 5개에 없지만 데이터로 답할 수 있는 질문 하나를 만들어 답하세요. 예시:
- 특정 배급사(CJ, 롯데, 메가박스)의 시장 점유율 변화
- 7년 모두 박스오피스에 등장한 영화 — 어떤 영화들이고 왜?
- 일본 애니메이션의 한국 흥행 추세
- 상영횟수 대비 관객수가 가장 좋은 영화

자유 질문이 평가에서 차별화 포인트입니다. 본인의 관심사를 진지하게 던지세요.

## 자주 막히는 부분

수업에서 한 번 짚어주지만 본인이 마주칠 가능성이 높은 것들:

1. **한글이 깨져요** → 시작 노트북 환경 설정 셀에 폰트 설정 있음. Colab이면 `!apt-get install -y fonts-nanum && fc-cache -fv` 후 런타임 재시작.
2. **`개봉일`이 문자열이라 월 추출이 안 돼요** → `pd.read_csv(..., parse_dates=['개봉일'])`로 읽거나 `pd.to_datetime` 한 번 거치기.
3. **`groupby` 결과가 Series인지 DataFrame인지 헷갈려요** → `.reset_index()` 하면 DataFrame이 됩니다.
4. **그래프 X축 라벨이 겹쳐요** → `plt.xticks(rotation=45)` 또는 `figsize` 가로 늘리기.
5. **결측치 있는 행에서 연산이 NaN이 나와요** → `dropna()`로 빼거나 `fillna()`로 채우기. 어느 쪽인지 본인이 정하기.
